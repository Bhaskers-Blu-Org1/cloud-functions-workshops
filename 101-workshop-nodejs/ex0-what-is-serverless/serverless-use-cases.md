# Where can can be applied?

## Serverless is the ultimate Microservice framework

If you have already have moved towards deconstructing your monolithic applications to embrace microservices and reactive programming concepts, then Serverless should be viewed as the ultimate evolution of a framework that delivers on these concepts.
